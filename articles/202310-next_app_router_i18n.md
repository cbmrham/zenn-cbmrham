# email-next-app-router

## はじめに

Next.js でメールを送信するアプリを作成する際に、メール送信後に別のページに遷移させたいという要件がありました。
Next.js のルーティングはクライアントサイドで行われるため、メール送信後にサーバーサイドでのルーティングを行う必要があります。
今回は、Next.js のルーティングをサーバーサイドで行う方法を紹介します。
